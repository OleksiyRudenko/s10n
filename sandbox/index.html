By default a preset value is inserted as a text input and textarea form elements value.
      Then those values get sanitized.
      This means that the browser has a chance to conduct basic sanitation
      (e.g. strips newlines for text input).
      This however depends on a particular browser in use.
    

    

      In BackEnd Emulation mode text inputs are updated as well,
      yet presets are fed to s10n directly.
      This emulates situation when a user sends form data to the
      backend directly, by means other than the browser form submission
      mechanisms.
